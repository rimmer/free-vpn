# FreeVPN

Stay private with you favorite sites safely with maximum security. Ublock sites and services, blocked in your country, like VK, Youtube, Linkedin or adult.

## Install

	$ npm install

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
